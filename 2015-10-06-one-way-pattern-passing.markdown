Definitions:

* **node** - A unit of executable code, probably focused around a particular subject.
* **message** - A JSON object
* **queue** - A message queue which receives messages from anywhere and broadcasts them to all nodes.
* **pattern** - A set of properties with concrete or abstract values. Used by nodes to test incoming
messages to determine if the node is interested in the message.
